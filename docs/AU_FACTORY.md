# 澳洲肉品工廠查詢系統

## 資料來源

- **PDF 檔案**: `AU-經我國核准之澳洲肉品工廠名單-系統認證-6-Feb-2026.pdf`
- **更新日期**: 2026年2月6日
- **檔案大小**: 約 930 KB

## 功能說明

### 搜尋功能
- 支援廠號搜尋（如：291, 292）
- 支援工廠名稱搜尋
- 支援地區搜尋（如：Victoria, Queensland）

### 篩選功能
- 全部工廠
- 牛肉工廠
- 羊肉工廠
- 混合工廠（羊肉/牛肉）

### PDF 連結
- 點擊「開啟 PDF」按鈕可直接查看完整名單
- PDF 包含所有經我國核准的澳洲肉品工廠資訊

## 資料欄位

| 欄位 | 說明 |
|------|------|
| 廠號 | 澳洲工廠認證編號 |
| 工廠名稱 | 英文官方名稱 |
| 所在地 | 澳洲州別 |
| 肉品類型 | 牛肉/羊肉/混合 |

## 技術實現

- 前端搜尋：即時過濾，無需後端 API
- PDF 檢視：瀏覽器內建 PDF 檢視器
- 響應式設計：支援手機、平板、桌面
